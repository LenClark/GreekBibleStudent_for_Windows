1 General Information about Bible Reader

The aim of Bible Reader is to provide you with a version of the Greek New Testament and some tools for exploring it on your own computer, This means that you can use it even when the internet is not available. It also means that you can work at your computer's speed and not that of your internet connection.

We'll introduce you to the different parts of the application in detail later but let's get an overview now of what Bible Reader offers. It provides

  • The New Testament text, displayed for a selected chapter
  • A parse window: simply by clicking and "Analysing" a word, you can get a grammatical analysis of that word;
  • At the same time, it will also try to give you details of how the word was used (as provided by the Liddell & Scott Intermediate lexicon);
  • A search facility allows you to get a list of all the verse where the word is used, together with some additional sophisitcation in search criteria;
  • It also provides a feature that allows you to create, save and restore notes for each verse in the New Testament.

So, let's look at the application in more detail.

2. The New Testament Text

When you first start the application, it will look something like this:

Let's describe the different parts of this window:

2.1 The main Text Area

It's pretty obvious what happens here.

We need to point out that the text is based on a version that is available in the public domain (based on the text provided by the Society of Biblical Literature - home site: https://www.sbl-site.org - and Logos Bible Software - home site:https://www.logos.com - see http://sblgnt.com), together with the parsing information. This has a number of implications:

  1. The text is essentially based on Westcott & Hort and may therefore display some differences from published versions of the Greek New Testament. (Note: for those who are wedded to the Textus Receptus, this is a critical text.) Where differences occur, they are often simply questions of variant readings.
  2. Since the parsing has been a labour of love by James Tauber (http://jktauber.com, see also http://jtauber.com), there could be errors - and, anyway, all parsing is likely to involve a degree of interpretation where there are any complications and we can never abdicate from our own responsibiity to engage with the text.

If you click on a verse in the text area, the pull-down Verse list will change to make that the current verse (of which, more later).

2.2 Selection of Book and Chapter

The first to "Combo Boxes" at the top of the window allow you to select specific books of the bible and a specific chapter within the selected book. When you select a different book from the one currently displayed, it will take you to the first chapter of the new book.

2.3 Selection of Verse

The third Combo Box allows you to change the current verse. This doesn't actually change anything visually but the significance is for the creation of notes: any note you create will apply specifically to the verse referenced by this Combo Box.

If you have a note displayed for your current verse and you change this verse reference, that note will disappear from view. (Don't worry! It will be conserved.) Similarly, if the verse you change to already has a note, then this note will now be displayed.

You can change the verse reference by selecting your target verse in this Combo Box or by clicking on the verse in the main text area.

2.4 The Notes Area

Any notes you create (or install) will be displayed in the area at the bottom half of the right hand side. If you want to create a note for a specific verse, type your note here. If you want to add Greek text to any part of your note, you can click on the Show Keyboard button to display a keyboard that will allow you to do so. (Of course, you can use multiple Microsoft keyboards to do the same thing, but it is difficult to create suitable accents and diacritics using provided keyboards.)

2.5 Analysis

This is a convenient diescription for the action of creating:

  1. Parse details, and
  2. lexical information for the selected word.

In fact, you only need to right click on a word and then select Analyse from the resulting pop-up menu to obtain parsing details and lexical information for that word.

Note: if you select multiple words, Bible Reader will provide a warning and will not be able to display the information.

The parsing details will be provided in the area in the top half of the right hand side. If you want to see the lexical information, you will need to click on the tab entitled Lexical Information. Note also that parse details and lexical information will remain available until you request a new analysis and that changing tabs does not erase data.

2.6 History

Bible Reader keeps a track of passages you have selected. If you want to return to a previous Book and Chapter, simply click on the little down arrow of the History Combo Box and select the passage you want.

Although not strictlly history, you can also select the previous chapter by clicking on the left arrow or the next chapter by clicking on the right arrow .

3 Additional Information about the Main Window

3.1 Getting the Window the way you want it

You can resize the window by dragging the window edges or corners in any desired direction. We will assume that you are familiar with resizing windows (and click-and-drag actions, in particular) and, if you are not, we recommend that you read a book on using your operating system:

The advantages of making the window bigger are:

  • The main text may be easier to read;
  • You will see more of the text if the chapter is a long one;
  • It may make many of the text boxes and buttons less cluttered and easier to use.

The advantages of a smaller window may be:

  • That the application better fits a small screen;
  • You may wish to have other applications open at the same time and this makes it easier to navigate around your estate.

What is important about the resizing of your window is that Bible Reader will remember your settings and return the window to the same size next time you use the program.

The window is divided into two halves and, between the main text area and the collection of small panes on the right hand side, there is a Splitter Bar. Slowly move your mouse from the main text area across to the right and, as you cross the divider, the cursor shape should change. At the point the curser has changed you can click and drag the bar with your mouse. The benefit of this change is that you can maximise the area in which you have main interest.

The application will remember the relative sizes of the two halves of the window as well when you return to the application.

3.2 Data used by Bible Reader and Relocation

The application depends on data in four directories (or folders) and it is probably worthwhile your knowing this. Immediately after installation, all four folders will be found in the directory:

C:\Users\yourUserName\AppData\Local\NTReader

where yourUserName is the name you registered when you set up your computer. (If someone else manages your computer, talk to him or her about this location and how you can find it.) These files and folders have been placed in an area for which you alone are responsible.

a) Configuration

Default initialisation data (such as the positions and sizes of the window and Splitter that we have just mentioned) will be stored in the Registry.

b) Source

This directory contains most of the data that the application uses. This includes:

  • the source files for the New Testament text;
  • the Liddell & Scott lexicon;
  • various files supporting the lexicon that we have yet to discuss;
  • other files that help manage and control the functioning of the application.
c) Notes

This is where your notes are stored. By default, your notes will be stored in a folder called default but we will explain in a moment how you can change this name and store muliple, separate sets of notes.

It is vital that you do not try to edit these files directly. If you do so, you may completely lose your notes.

d) Help

This file and associated images.

3.3 Changing the Location of your Source Files, Notes and the Help files

It is possible to change the location of your source files. This is probably only of value if you have multiple hard disks on your computer system and want to save space on your default drive.

To make this change, select the following option from the application menu (at the top of the computer screen):

File | Change Location of Application Files

Selecting this option will open a small dialog that will allow you to navigate to the place on your disk where you want to store the data:

Choose the new location (i.e. the directory where you want the Source folder to go). Note that Bible Reader will move the Source folder itself, as well as all the files it contains, so you do not need to create a folder called "Source" or "Help" or "Notes" . You can create any other directory you need by clicking on the New Folder button and following directions.

The various files in the Source directory must be available to the application when it runs, so do not move it to a location that might not be available. (For example, avoid external hard disks or flash memory drives.)

4 Notes

You may want to keep notes for a variety of reasons. For example, you may want to note text variants for each verse. Alternatively, you may want to make exegetical or critical notes against a verse as you investigate it. Bible Reader allows you to do this.

Note that the notes area is not a word processor: it provides very limited formatting capability. One important feature it does provide is the ability to enter Greek text as well as your normal text. Simply click on the Show Keyboard button and a simple Greek keyboard will be displayed:

4.1 The Keyboard

This keyboard is used for both the creation of notes and entering details into the search criteria. Because of this, you can change the target location of anything you type. This explains the Radio Buttons to the right of the keyboard. Make sure the option Notes Editor, is selected when using it to enter notes:

Note that, while the keyboard is displayed, the button that displayed it has changed to Hide Keyboard.

In practice, a lot of the time you may enter Greek text by cutting and pasting from the main text. Using the virtual keyboard allows you greater freedom in entering information about Greek. Note that the accents and diacritics on the right side of the keyboard are entered after the letter to which they are attached. In other words, to type ἤ, first type eta (η), then the breathing, then the accent. If you are not sure which symbol is which, simply hover the mouse over the key and a description of the key will be displayed for a short time.

4.2 Menu Options

(We shall refer to a set of notes as a Notes Set. You can have multiple Notes Sets but you will only be able to access one Notes Set at a time.) You can change the Notes Set that you use in the following ways:

4.2.1 Create a New Notes Set

From the main menu at the top of the screen, select:

File | Manage Notes | Create a New Notes Set

When you select this option, the following dialog will be displayed:

Provide a name for your new Notes Set. For example, MyComments.

This name will be used as the basis for the name of the file in which your notes will be saved, so it is recommended that you avoid using spaces. Any new notes you create in this Notes Set will be saved in a file with the same name and with the extension .note.

When you restart Bible Reader, the application will reload the notes set that you were using at the end of your previous session. For example, if you created a new Note Set called MyComments, the notes will be saved in a file called MyComments.note.

4.2.2 Choose a different Notes Set

You can change to a different, existing Notes Set by selecting the menu option:

File | Manage Notes |Choose a different Notes Set

This will open the following dialog:

Note the small down arrow to the right of the text entry box. By clicking on this you will see a list of all available Notes Sets. Select the Set to which you want to change and then click on the OK button.

4.2.3 Delete an Existing Notes Set

If you want to remove a Notes Set from the system, select the following from the main menu:

File | Manage Notes | Delet an existing Notes Set

This will open a dialog that is identical to that of §4.2.2. The only difference is that the current Notes Set will not be in the list available. This is because you cannot delete the Notes Set currently in use.

Please note that this action is irrevocable: once a Notes Set has been deleted, you cannot recover it.

4.2.4 Reminding you of the Name of your current Notes Set

The final menu option under this heading simply displays the current Notes Set name. It doesn't actually change anything.

5. The Grammatical Summary

To parse a word, right click on the word of interest and then select Analyse Word from the menu that pops up.

If you have previously changed to the Lexical Information or Search Results window, selecting the Analyse button should switch the top right pane back to the Grammatical Summary.

6. Lexical Information

6.1 The Main Tool

This is used in tandem with the Grammatical Summary facility. In other words, Analyse means both parse the word and obtain lexicon information, if possible. You will always have to switch to the Lexical Information pane to see the meaning of the selected word:

We have tried our best to ensure that a meaning for all words has been provided but the root form identified by James Tauber is not always that identified in Liddell & Scott. Please note also that the edition of the lexicon in the public domain (and, hence, used in Bible Reader) is fairly old and modern criticism may throw the provided meanings of some of the less well used words into question.

It would be helpful to advise us where lexical information is in error or missing.

6.2 Appendices

You will note from the tab in the lower right-hand pane that we have provided the Appendices to Liddell & Scott's lexicon in separate pages. The particular value is that these are referred to in the body of the lexicon in abbreviated form and you can check for the expanded meaning in the Appendices. Unfortunately, at the moment, there is no facility for searching the Appendices for a given abbreviation.

The Appendices are broken into the following areas:

  • Authors and Works
  • Epigraphical Publications
  • Papyrological Publications
  • Periodicals
  • General Abbreviations

These are as provided by Liddell & Scott.

You will see that these can be selected using the tabs at the bottom of the pane:

7. Search

Clicking on the Search tab on the top of the lower right pane will reveal the entry panel for the search facilty:

This is the simple search facility and we shall discuss more advanced features later.

7.1 Performing a Simple Search

The purpose of the simple search feature is to supply a word (in the Search for a word box) and search for all occurrences of the word in the New Testament.

There are two ways of entering suitable words in the Search for a word box:

  1. You can select and copy a word in the main text area and then paste it into the box, or
  2. You can open the keyboard (by clicking on the Show Keyboard button), select the target area as "Primary Search String" and type the word in manually. A word of warning: NTReader can only find the selected word if it exists in the New Testament either:
  • exactly as typed, or
  • if you have typed the root of the desired word

In other words, text typed into the Search fields manually may result in no matches.

7.1.1 Using Copy and Paste

This is probably the easier method and is likely to be the means that you will use. There are three ways in which you can do this:

  1. Select a word in the main text area and then right click on the selected word (i.e. click once on the right button of your mouse). You should then see a small pop-up menu that includes various Copy options. Click on the Copy Current Word to ... | Memory (i.e. single click with the left mouse button). Now move your mouse to the Search for a word box and click in the box (using the left mouse button). Once your cursor is positioned in the box, right click (click the right mouse button once) and select the Paste option.
  2. Select a word in the main text area and then, while holding down the Control key (Ctrl), press the key C. Now move to the Search panel and click your mouse in the Search for a word box. Once your cursor is positioned in the box, hold down the Control key again and press the key V.
  3. Right clickon the word in the main text area (as in option 1) and then select Set Search Details | Copy Current Word to Search from the pop-up menu.
7.1.2 Using the on-screen Keyboard

If the keyboard is not already visible, click on the Show Keyboard button. The following Keyboard panel should rise at the bottom of the left-hand side:

Click on the Main Search Text Radio Button to make sure that anything you type goes into the right place. Then type your Greek word.

If a vowel has a breathing, accent or other diacritic, type the naked letter first and then each of the additional symbols. If you're not sure which key will provide the correct symbol, simply hover the mouse over the key briefly and a description of the key should appear.

7.1.3 Acting on the Provided Word

Once you have entered a word in the Search for a word box, click on the Search button (at the left side of the panel). This should open the Search Results pane at the top right of the window and display the results of the search. Be patient: it could take a little time.

Note: by default, you can enter any form of a given word and it will search for all occurrences of the root word, no matter what form it takes. So, a search for κτίσεως (for example) will also return verses containing κτίσις, κτίσει, and so on.

If you want to search for a specific form of the word (e.g. κτίσεως and only κτίσεως), then change the Radio Button on the Search pane to Exact match.

7.2 The Advanced Search

Click on the Advanced Search button (on the Search pane - bottom right) and additional details will appear in the pane:

We have called the text box following the words "words of", the Secondary Search Text. You enter a word here in much the same way as before using either 1 or 2 of §7.1.1 or the virtual keyboard (§7.1.2) with the Radio Button changed to Secondary Search Text. The Radio Button options on the right-hand search pane function much as before.

So, the point of the advanced search is that you can limit the search results to the occurrences of a word that occurs within the context of a second word.

7.3 What else can you do with search results?

7.3.1 Clicking on a Search Result

If you click on a specific Search Result, the main text will take you directly to that Book and Chapter.

7.3.2 Copying the Search Results

Selecting the following main menu option:

Edit | Search and Results | Copy Selected Search Result | Memory (or Notes)

will copy the full contents of your Search Results into memory. You can then paste this into your Word Processor or other application of choice. Alternatively, you can right click on the relevant Search Result and select the relevant option from the pop-up menu.

8 Adjusting the right-hand Pane

The one control on the top tool bar that we haven't discussed is the pull-down list on the extreme right:

The purpose of this feature is to enable you to easily manage the information in the various sections on the right-hand side of the application. Sometimes you may want to concentrate on information in the top panel (e.g. Search Results) and at other times you may want to focus on the lower panel (e.g. if creating a complex note). This feature will enable you to resize these sub-panels in a controlled fashion.

The options availabel are:

8.1 Minimise Bottom

In other words, reduce the bottom panel to an absolute minimum and allow you to focus on the top panel. To use this option, simply click on the down arrow beside the Resize control and select the Minimise Bottom option:

8.2 Minimise Top

This has a similar effect but works in the other direction.

8.3 Equalise Both

This will restore the dividing bar between the two sub-panels to the mid-point vertically. This means that both the top and bottom panel on the right hand side will be roughly the same size.

You are still able to resize these panels manually by clicking on the Splitter Bar and dragging it into the desired position:

Note that some of the menu options are provided by Apple and are not under the control of the application. We shall only discuss those that are specific to Bible Reader.

9.1 Manage File Locations

The option for relocating key files (all Source data, Notes and Help) is found on the main menu:

File | Change Location of Application Files

This feature was discussed in §3.3.

9.2 Manage Notes

All options for managing notes were discussed in §4.2.

9.3 Copying the Bible Text

9.3.1 Copying the whole of the current Chapter

You can copy the contents of the whole of the current chapter to the pasteboard from the main menu using the following sequence of options:

Edit | Bible Text | Copy Current Chapter to ...

This provides to final possible destinations for the copy process:

  • Memory - i.e. to the clipboard, so it can be inserted into any text-based application using the paste option
  • Notes - so that the selection is copied directly into the currently open notes page.

If copying to memory, once the copy process has been completed, you will be notified by a simple alert. The text can subsequently be pasted into any application of your choice that allows you to do so (e.g. a Word Processor).

If copying to notes, the selected text will be inserted at the current cursor position.

Of course, you can also perform a copy manually by selecting all the text in the window and then using the Copy facility.

9.3.2 Copying the Current Verse

This will copy the verse from the main text area in which the fine cursor (or Caret) is positioned. To do this select:

Edit | Bible Text | Copy Current Verse to ...

This functions in much the same way as copying a chapter (see §9.3.1)

9.3.3 Copying the current Word

This copy process is similar to the previous two but is very specific and will copy a single word. To do this use:

Edit | Bible Text | Copy Current Word to ...

This functions in much the same way as copying a chapter (see §9.3.1)

9.3.4 Copying the current Word to the Search Text Field

To do this, use:

Edit | Search and Results| Copy Word to Primary Search

Selecting this option will copy the current word directly to the main search text field in the Search panel:

If the Search pane is not already visible, using this option will insert the selected word and make the search option visible.

9.3.5 Copying the Current Word to the Secondary Search Text Field

To do this, use:

Edit | Search and Results| Copy Word to Secondary Search

This will copy the current word to the secondary text field:

If the Secondary search option is not currently visible, this will make it visible and active.

9.4 Copying Text from the Grammatical Summary and Lexical Information Panels

9.4.1 Copying the Grammatical Summary

The whole of the current Grammatical Summary will be copied to the pasteboard using the menu options:

Edit | Handle Analysis | Copy Grammatical Summary to ...

As described in §9.3.1, the final choice in this menu option will be to either copy the selection to the clipboard (available for pasting into any text-based application) or directly into the current note.

9.4.2 Copying the Lexical Information

The whole of the Lexical Information text can be copied in much the same way as §9.4.1 using:

Edit | Handle Analysis | Copy Lexical Summary to ...

9.4.3 Copying both the Grammatical Summary and the Lexical Information

Both the above sections can be copyied at the same time using this option. Select:

Edit | Handle Analysis | Copy Both to ...

9.5 Copying Search Results

The full results of a search can be copied to the pasteboard or current note (as described in §9.3.1). Use

Edit | Search and Results | Copy All Search Results

and

Edit | Search and Results | Copy Selected Search Results

The first option will copy all the results for the current search; the second will copy only the result in which the cursor is currently located.

9.6 Changing Text Sizes and Colour and Panel Background Colour

You can change the font size and text colour separately for each of the following:

  • The main text area
  • The Grammatical Summary
  • The Lexical Information
  • Search Results
  • Notes

You can also change the colour of the background in each of these areas.

You may want to use a larger font size if you find the text a little difficult to read. Equally, you may want to reduce the font size to fit more into the space available.

The ability to change text and background colours may simply satisfy your esthetic preferences but also some people with reading difficulties find that changing these colours in a controlled way can make reading easier.

All three types of change are controlled by a single menu option:

File | Preferences

This will open the Font Management dialog:

Attributes for each panel can be accessed separately by clicking on the relevant radio button at the left of the dialog.

A list of standard font sizes is provided in the pull-down list for the fonts but you can enter any sensible value (including fractional font sizes). The colours can be selected using one of a number of colour selection tools.

Clicking on OK will close the dialog and activate any changes you have made for all of the five areas. Clicking on Cancel will return you to the main application without making any changes and any changes that you made in the dialog will be lost.

Any changes that you make will be saved and, next time you start the application, these changes will be retained.
